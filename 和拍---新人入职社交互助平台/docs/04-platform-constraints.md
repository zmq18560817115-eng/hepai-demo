# 04 · 载体与约束（钉钉插件）

## 目标载体


| 项    | 约束                                                      |
| ---- | ------------------------------------------------------- |
| 运行环境 | 钉钉工作台 · 企业内部 H5 微应用                                     |
| 视口   | **375 × 812**（iPhone 逻辑像素）为设计基准；原型外壳 `max-width: 390px` |
| 安全区  | 预留顶部 44px（钉钉标题栏）、底部 34px（Home Indicator）                |
| 网络   | 企业内网为主；关键路径需 **骨架屏 + 离线提示**（二期）                         |


## 权限审计（metadata.json）


| 权限            | 原型是否需要        | 说明                        |
| ------------- | ------------- | ------------------------- |
| `camera`      | **否（MVP 移除）** | 无头像拍摄流程，用默认头像 / 钉钉头像      |
| `microphone`  | **否**         | 无语音功能                     |
| `geolocation` | **暂缓**        | 午餐「地图」为示意 UI；真实 LBS 匹配放二期 |
| `GEMINI_API`  | 演示可选          | 仅 AI 文案增强时使用，非核心路径        |


> 正式上架前应更新 `metadata.json`，去掉未使用权限，避免钉钉审核驳回。

## 布局原则（原型已遵循）

1. **单栏纵向滚动**，避免桌面多列（`md:grid-cols-`* 在插件内降级为 1 列）。
2. **顶栏精简**：Logo + 角色切换；通知/头像可收起到二级。
3. **底栏固定 3 项**，触控热区 ≥ 44px。
4. **Neo 视觉**保留，但字号在窄屏使用 `text-sm` 基准。

## 弱网 / 空状态（占位文案）


| 场景   | 文案                 |
| ---- | ------------------ |
| 加载中  | `正在同步你的安全屋…`       |
| 失败   | `网络有点挤，点一下重试`      |
| 无导师  | `带教老师还没就位，HR 正在配置` |
| 匹配超时 | `饭搭子还在路上，稍后再试`     |


## 原型 vs 正式版差异


| 原型               | 正式钉钉版                      |
| ---------------- | -------------------------- |
| 浏览器 + 手机框演示      | `dd.biz.navigation` 隐藏系统导航 |
| localStorage 存状态 | 后端 Session + 钉钉 userId     |
| 顶栏三角色切换（演示用）     | 钉钉身份自动识别角色                 |
