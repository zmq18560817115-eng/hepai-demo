# 03 · 信息架构与页面清单

> 仅描述 UI 结构与导航，API 细节见 `SYSTEM_FRAMEWORK_DOCUMENT.md`。

## 站点地图

```
和拍（钉钉 H5 插件）
├── C 端 · 新员工
│   ├── P0 入职盲盒（仅首次 / 可重置演示）
│   ├── P1 安全屋（首页）
│   ├── P2 蹭饭地图 / 午餐匹配
│   ├── P3 带教导师（列表 → 详情占位）
│   ├── P4 我的工位（贴画奖励桌面）
│   └── P6 企业 AI HR（悬浮入口 → 全屏对话）
│       ├── 制度层：企业管理准则（请假/报销/入职/绩效/保密/带教）
│       └── 融入层：ai-hr-onboarding Skill（面具/焦虑/午餐/导师）
├── C 端 · 导师
│   └── P5 导师工作台（新人列表 + 状态）
└── B 端 · HR
    └── P7 HR 仪表盘
```

## AI HR 信息架构（制度 × 融入）


| 层级  | 知识源                                | 典型问题             | 回答要求                   |
| --- | ---------------------------------- | ---------------- | ---------------------- |
| 制度层 | `enterprise-management-guidelines` | 年假怎么请、报销标准、试用期考核 | 引用准则条款 + 办事入口（钉钉人事/费控） |
| 融入层 | `ai-hr-onboarding`                 | 面具、焦虑、蹭饭、导师话术    | 结合用户能量/面具/导师数据         |
| 整合层 | 带教准则 § 和拍对照表                       | 制度与和拍说法不一致       | **以企业管理准则为准**，和拍为辅助    |


后端：`GET /ai/hr/guidelines` 浏览准则目录；`POST /ai/hr/chat` 返回 `citations[]`。

## 导航规则


| 角色       | 顶栏切换 | 底部导航                 | 盲盒         |
| -------- | ---- | -------------------- | ---------- |
| newcomer | 新人   | 安全屋 / 蹭饭 / 导师（盲盒完成后） | 未完成前全屏，无底栏 |
| mentor   | 导师   | 无（单页工作台）             | 跳过         |
| hr       | 管理端  | 无                    | 跳过         |
